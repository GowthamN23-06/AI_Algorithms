The Project is Initial development stage of routing the Network traffic from Point A to Point B located exactly opposite  The Minimun number of spanning satellites required is optimised using Ksukal algorithm or Wlelsh Powell Algorithm based on number of satellites and distance metric.
The metric is calculated using the amount of traffic and optimized using the GBR calculations.
3GPP R18 is used for this initial development and all programs are implemented in C++, Python libraries.
